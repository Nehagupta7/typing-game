Key//Strike

A typing arcade game — type falling words before they cross the danger line.
Single HTML file, just open it in a browser, no install needed.

How to play

Just start typing — it locks onto the matching falling word. Backspace fixes
a mistake, Esc pauses. Needs a physical keyboard.

Modes


Survival — 4 lives, gets faster over time, ends when you run out of lives
Timed — 60-second score attack


Difficulty

Easy (everyday words) → Medium (+ workplace vocabulary) → Hard (+ long words)
→ Warrior (+ dev/coding tokens)

Notes


No backend — no accounts, no saved scores between sessions.
Needs an internet connection the first time you open it (for fonts).
